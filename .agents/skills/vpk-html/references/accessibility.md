# Accessibility Rules

All templates must provide:

- Semantic landmarks where appropriate.
- Logical heading order.
- Visible focus states in light and dark modes.
- Keyboard access for every meaningful control.
- Alt text or accessible names for meaningful images and SVGs.
- Decorative assets marked with `aria-hidden="true"`.
- Contrast-safe token pairs in light and dark modes.
- `prefers-reduced-motion` handling.
- Responsive layout down to mobile widths.
- Print styles that avoid clipped or unusable content.
