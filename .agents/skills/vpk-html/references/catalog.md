# vpk-html Catalog

Every template is addressable by its clean template id in the render payload.
Templates share the same offline shell, embedded font handling, source rendering,
dark-mode toggle, print baseline, and accessibility helpers.

## Exploration and Planning

| Template | Use |
| --- | --- |
| `exploration-code-approaches` | Compare code approaches, tradeoffs, and recommended path. |
| `exploration-visual-designs` | Compare visual directions and design variants. |
| `implementation-plan` | Milestones, risks, validation gates, and ownership. |

## Diagram Primitives

Supported primitives are `architecture`, `flowchart`, `quadrant`, `bar-chart`,
`line-chart`, `donut-chart`, `state-machine`, `timeline`, `swimlane`, `tree`,
`layer-stack`, `venn`, `candlestick`, and `waterfall`.
