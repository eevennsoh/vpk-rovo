# Source Policy

- If the user supplies complete material and asks only to render it, transform
  only that material.
- Verify current facts, product/company/person/version details, financial
  claims, release dates, and latest/recent claims against primary sources before
  rendering.
- Render a visible Sources section when browsing or research is used.
- Render a visible Credits section when third-party copied or adapted material
  requires attribution.
- Prefer supplied material and vendored Algebrica material for math pages.
- If sources conflict or cannot be checked quickly, ask one focused question.
- Metadata comments may include template name, vpk-html version, render
  timestamp, and source labels or basenames. They must not include absolute
  source file paths by default.
